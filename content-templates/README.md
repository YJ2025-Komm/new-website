# Content Template System for GeoRankers

## How It Works

1. **Fill out the template** - Use `blog-post-template.md` as your starting point
2. **Add your images** - Upload images to the `attached_assets/` folder in your Replit
3. **Share with me** - Send the completed template and I'll implement it on your site
4. **Quick updates** - For future posts, just fill out the template again
